# SafeSnap Documentation Overview

This folder contains key documentation files for the SafeSnap Chrome Extension project. Each file serves a specific purpose to help developers, contributors, and users understand the design, architecture, and detection strategies of the extension.

## File Summaries

- **ARCHITECTURE.md**
  - Provides a high-level overview of the SafeSnap system architecture, including major components, data flow, and extension structure.

- **DESIGN_SPECIFICATION.md**
  - Details the product's design, including feature set, detection and replacement strategies, UI/UX specifications, error handling, and key design decisions.

- **DEMONSTRATION.md**
  - Offers a walkthrough of SafeSnap's features and workflows, with screenshots and step-by-step usage instructions for end users.

- **DETECTION.md**
  - Explains the PII detection engine in depth, covering dictionary-based and pattern-based detection, scoring signals, and technical implementation details.

---

For more information on contributing or using SafeSnap, see the main README.md in the project root.
